# 管理员管理模块

## 功能概述

管理员管理界面是用户管理的简化版本，专为超级管理员设计，方便快速管理系统管理员用户，无需在用户管理界面中筛选。

## 功能特性

- ✅ 管理员列表分页展示
- ✅ 用户名搜索
- ✅ 查看管理员详情
- ✅ 编辑管理员信息
- ✅ 重置管理员密码
- ✅ 显示管理学院权限范围
- ✅ 显示管理员角色信息

## 权限要求

- `admin-user:page` - 查看管理员列表
- `admin-user:details` - 查看管理员详情
- `admin-user:update` - 编辑管理员信息
- `admin-user:reset-password` - 重置管理员密码

## 技术实现

### API模块 (`/src/api/modules/admin.ts`)

- 完全使用生成的接口定义
- 从 `@/api/generated/.ts.schemas` 导入类型
- 提供类型安全的API方法

### 视图组件 (`/src/views/AdminManagement/index.vue`)

- 遵循项目现有代码规范和风格
- 使用 Element Plus 组件库
- 响应式设计，支持不同屏幕尺寸

### 路由配置

路径：`/admin-management`
权限：`admin-user:page`

## 与用户管理的区别

| 特性         | 用户管理 | 管理员管理   |
| ------------ | -------- | ------------ |
| 目标用户     | 所有用户 | 仅管理员用户 |
| 功能复杂度   | 完整功能 | 简化功能     |
| 分配角色     | ✅       | ❌           |
| 编辑信息     | ❌       | ✅           |
| 显示管理学院 | ❌       | ✅           |

## 使用说明

1. 确保拥有 `admin-user:page` 权限
2. 访问菜单中的"管理员管理"
3. 可通过用户名搜索特定管理员
4. 点击操作按钮查看详情、编辑或重置密码
